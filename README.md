# Mini Project Management Portal

A full-stack task manager built for the o2h hiring assessment.

## Tech Stack
- Frontend: React.js, Axios, Custom CSS (Responsive, Dark Mode)
- Backend: Node.js, Express.js
- Database: MongoDB

## Setup Instructions

### 1. Prerequisites
Ensure you have Node.js and MongoDB installed and running on your local machine.

### 2. Backend Setup
```bash
cd backend
npm install
node server.js
